StretchSenseClass

The StretchSense library with all the fonction to paired the StretchSense Fabric Sensor to your iOS application

Note

On the StretchSenseClass you can several list of peripheral, the number of sensor, the values of the sampling time and filtering for the sensor and some feedback informations
