    StretchSenseClass

    The StretchSense library define all the fonctions to paired the StretchSense Fabric Sensor to your iOS application

    Author

    Jeremy

    Copyright

    2016 StretchSense

    Date

    19/05/2016

    Version

    1.0.0

    Note

    On the StretchSenseClass you can find several list of peripherals (available, connected, saved), the number of sensors, the settings values (number of sample to display, sampling time, average filtering value) and the feedback informations

    //Exemple
class ViewController: UIViewController {

   var stretchsenseObject = StretchSenseClass()
   override func viewDidLoad() {
       // This function is the first function called by the View Controller
       super.viewDidLoad()
       // Init the StretchSense Library and Bluetooth
       stretchsenseObject.startBluetooth()
       // Start Scanning new peripheral
       stretchsenseObject.startScanning()
   }

   @IBAction func connect(sender: AnyObject) {
       stretchsenseObject.startScanning()
       // Get all the peripheral available
       var listPeripheralAvailable = stretchsenseObject.getListPeripheralsAvailable()
       // Explore all the peripheral available
       for myPeripheralAvailable in listPeripheralAvailable{
           // Connect all the peripheral available
           stretchsenseObject.connectToPeripheralWithCBPeripheral(myPeripheralAvailable)
           print(myPeripheralAvailable)
       }
   }

   @IBAction func printValue(sender: AnyObject) {
       // Get all the peripheral connected
       var listPeripheralConnect = stretchsenseObject.getListPeripheralsConnected()
       // Explore all the peripheral connected
       for myPeripheralConnected in listPeripheralConnect{
           // Print the value of all the peripheral connected
           print(myPeripheralConnected.value)
       }
   }
}
